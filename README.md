# Вершина — премиальный клининг в Сочи

Лендинг. React + Vite + TypeScript + Tailwind CSS.

## Концепция
Сайт продаёт не «услуги уборки», а уверенность: точная цена до приезда (калькулятор), прозрачный чек-лист 47 пунктов, гарантия 48 часов, до/после без постановки, отдельное направление для посуточной аренды. Всё общение — WhatsApp-first.

## Запуск
```bash
pnpm install
pnpm run dev      # разработка
pnpm run build    # сборка в dist/
```

## Где что лежит
- `src/data.ts` — ВЕСЬ контент: телефоны, WhatsApp, тарифы, ставки ₽/м², чек-листы, кейсы, FAQ
- `src/components/` — секции в порядке страницы (см. App.tsx)
- `public/images/` — изображения (сейчас плейсхолдеры, план замены в ASSETS.md)
- `source-assets/images/GPT/` — исходные GPT-генерации, не попадают в production build
- `preview.html` — статичное превью без сборки (открыть двойным кликом), в прод не брать

## Не забыть
- Заменить телефон/WhatsApp на реальные в `src/data.ts`
- Подложить реальные фото по ASSETS.md
- Цифра «сегодня убрано 1 240 м²» в Hero — задаётся руками или подключается к CRM
