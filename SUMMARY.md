# Вершина — summary проекта (для продолжения работы)

Обновлено: 12.06.2026

## Что это
Лендинг клининговой компании «Вершина» (Сочи, доступный premium). Полностью переписан с нуля — НЕ по шаблонному ТЗ, а по концепции «продаём определённость, а не услуги». Старый репозиторий github.com/radzun9080002222-ship-it/my-cleaning-peak подлежит полной замене содержимым папки `vershina-site/`.

## Концепция (важно понимать, прежде чем править)
Главный страх клиента премиум-клининга — не «плохо уберут», а «непонятно сколько стоит и кого я пускаю в дом». Поэтому:
- **Калькулятор с финальной ценой** (м² × тариф) — цифра сразу, без «менеджер свяжется». Бронь = кнопка в WhatsApp с подставленным текстом.
- **Манифест «Мы убрали из клининга всё, за что его не любят»**: цена до приезда / приёмка по чек-листу из 47 пунктов / 48 часов гарантии.
- **Чек-листы услуг открыты до заказа** (аккордеоны) — прозрачность вместо «качество и доверие».
- **До/после с интерактивным ползунком** + факты (часы, люди, район).
- **Отдельный B2B-блок для посуточной аренды** (слот 3 часа, фотоотчёт, абонемент).
- WhatsApp-first, форм и модалок нет.

⚠️ Обещания «47 пунктов», «страховка», «опоздание — скидка», «48 ч гарантии» — заложены как стандарт бренда. Перед запуском подтвердить, что операционно выполнимы, иначе убрать.

## Стек и запуск
React + Vite + TypeScript + Tailwind CSS.
```bash
cd vershina-site
npm install
npm run dev    # разработка
npm run build  # прод-сборка в dist/
```
`preview.html` — статичная копия сайта без сборки (открыть двойным кликом). Для быстрого просмотра, В ПРОД НЕ БРАТЬ.

## Структура страницы (= порядок в src/App.tsx)
Header → Hero («Чисто.» + 47/48ч/0₽) → Manifesto (стандарт) → Calculator (тёмный) → Services (аккордеоны с чек-листами) → Cases (до/после) → Process (4 шага) → Trust (фото команды + процедуры) → Rent (тёмный, посуточка) → Faq → FinalCta → Footer.
